# Offer-Catcher — AI Job Matching System

An AI-powered job matching tool: upload a PDF resume, get matched against a mock
database of top-tier company roles via vector keyword matching, and receive a
streaming AI "Resume-to-Job Matching Diagnosis & Optimization Report".

Bilingual UI (中文 / English), default language **Chinese**, persisted in
`localStorage`. The selected language is forwarded to the backend so the AI
report is generated in that language.
